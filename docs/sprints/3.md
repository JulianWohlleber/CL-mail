# Sprint 3 — Keyboard cheatsheet + Apple Mail preset

**Lead:** Priya (frontend), Tom (junior, polish)
**Tag:** v0.1.3

## Research

**Tom:** "Pressed `?` to see shortcuts. The ShortcutsHelp panel is there but
its layout is one long column — hard to scan. Mailspring users keep typing
`Cmd+E` for archive (which is the Apple Mail standard) and nothing happens.
Adding an `apple-mail` keymap covers a lot of accidental misfires."

**Priya:** "Also: shortcuts shown in the help panel are pulled from
superhumanKeymap directly, ignoring the user's actual preset. That's
misleading."

## Design

- Two-column shortcut grid grouped by category
- Add an `apple-mail` preset that mirrors macOS Mail.app conventions
- Help panel reads the *active* preset, not hard-coded superhuman

## Development

- New `apple-mail.keymap.ts`
- Wire it into the keymap registry
- ShortcutsHelp: read from settings.keymapPreset

## Testing

- Regression test #14: ShortcutsHelp doesn't hard-code superhumanKeymap
